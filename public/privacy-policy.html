🌙 몽글몽글 개인정보 처리방침

시행일: 2026년 3월 21일 | 최종 수정: 2026년 3월 21일

1. 수집하는 개인정보

몽글몽글(이하 "앱")은 서비스 제공을 위해 다음 정보를 수집합니다:

  • 자동 수집: 익명 사용자 ID (Supabase Auth 자동 생성), 앱 사용 이벤트 (꿈 기록 횟수, 기능 사용 빈도)
  • 사용자 입력: 닉네임, 꿈 기록 내용, 달이 대화 내용, 커뮤니티 게시글/댓글
  • 결제 정보: Apple/Google/Stripe를 통한 구독 거래 ID (카드 정보는 직접 수집하지 않음)

2. 개인정보의 이용 목적

  • 꿈 해몽 서비스 제공 및 개인화
  • 구독 상태 관리 및 권한 확인
  • 서비스 개선을 위한 익명 통계 분석
  • 고객 문의 응대

3. 개인정보의 보관 및 파기

  • 회원 탈퇴 시 즉시 삭제 (관련 법령에 따라 일부 보관이 필요한 경우 제외)
  • 꿈 기록 데이터: 사용자 요청 시 즉시 삭제
  • 결제 거래 로그: 전자상거래법에 따라 5년 보관 후 파기

4. 개인정보의 제3자 제공

앱은 다음 경우를 제외하고 개인정보를 제3자에게 제공하지 않습니다:

  • 법령에 의한 요청이 있는 경우
  • 서비스 제공에 필요한 외부 서비스: Supabase (데이터베이스), OpenAI (꿈 해석 AI)

5. 외부 서비스 이용

  • Supabase: 사용자 인증 및 데이터 저장 (AWS 서버, 암호화 전송)
  • OpenAI: 꿈 해석 AI 처리 (꿈 내용이 API로 전송되며, OpenAI 데이터 사용 정책에 따름)
  • Apple/Google: 앱 내 결제 처리

6. 이용자의 권리

  • 개인정보 열람, 수정, 삭제 요청 가능
  • 앱 내 MY 탭 → 데이터 초기화로 직접 삭제 가능
  • 문의: monggeul.app@gmail.com

7. 쿠키 및 추적

앱은 쿠키를 사용하지 않습니다. 앱 사용 이벤트는 익명으로 수집되며 개인을 식별할 수 없습니다.

8. 아동 개인정보

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.
