# 🌙 몽글몽글 — Brand Identity Guide

> Version 1.0 | 2026.03

---

## 1. 브랜드 개요

**몽글몽글**은 꿈을 기록하고, 달이와 함께 해몽하는 꿈 동반자 앱입니다.
사용자의 무의식을 따뜻하게 풀어주는 '밤하늘 아래 대화'를 컨셉으로 합니다.

### 브랜드 키워드
`몽환적` `따뜻한` `신비로운` `친근한` `밤하늘` `달빛`

### 브랜드 톤 & 보이스
- 전문가처럼 말하지 않되, 깊이 있게
- 친구처럼 편하되, 가볍지 않게
- 신비롭되, 무섭지 않게

### 색상 사용 원칙

1. **어두운 배경(Night/Deep) 위에 밝은 텍스트(Primary)** 사용을 원칙으로 합니다
2. **Purple Deep**는 CTA 버튼에만 사용하며, 배경색으로 사용하지 않습니다
3. **운세별 색상**은 해당 운세 컨텍스트에서만 사용합니다 (Amber=재물, Pink=연애 등)
4. 인쇄 시 CMYK 변환: Night → K100에 가깝게, Purple → Pantone 2665 C 참조

## 6. UI 컴포넌트 스타일

### 카드
- 배경: rgba(45, 40, 88, 0.5)
- 테두리: 1px solid rgba(166, 124, 239, 0.12)
- 모서리: 26px (--r-lg)
- 효과: backdrop-filter: blur(12px)
- 호버: 테두리 밝아지고, box-shadow 추가

### 버튼 (CTA)
- 배경: linear-gradient(135deg, #7C5CBF, #9B6DE0, #A67CEF)
- 모서리: 20px (--r-md)
- 그림자: 0 4px 20px rgba(124, 92, 191, 0.35)
- 효과: 자동 시머 애니메이션 (4초 주기)
- 호버: 리프트 + 그림자 강화

### 뱃지
- 모서리: 20px
- 크기: 11px, 패딩 3px 11px
- 길몽(bl): 금색 글로우
- 흉몽(bb): 빨강 글로우
- 연애운(bv): 핑크 글로우

### 입력 필드
- 배경: rgba(255, 255, 255, 0.05)
- 테두리: 1px solid rgba(255, 255, 255, 0.08)
- 포커스: 보라색 테두리 rgba(166, 124, 239, 0.4)
- 폰트: 16px (iOS 줌 방지)

## 7. 비주얼 효과 / Visual Effects

### 글로우 (Glow)
- 활성 요소에 은은한 빛 효과
- box-shadow + text-shadow 조합
- 예: `box-shadow: 0 0 10px rgba(166,124,239,.15), 0 0 20px rgba(166,124,239,.05)`

### 글래스모피즘 (Glassmorphism)
- 반투명 배경 + backdrop-filter: blur()
- 카드, 모달, 탭바에 적용
- `.glass { backdrop-filter: blur(20px); background: rgba(30,24,64,.6); }`

### 시머 (Shimmer)
- CTA 버튼에 자동 적용
- 4초 주기로 빛이 좌→우 이동
- 사용자 행동 유도

### 펄스 글로우 (Pulse Glow)
- 중요 알림이나 CTA에 사용
- 2초 주기 맥박 효과
- `.pulse-glow { animation: pulseGlow 2s ease-in-out infinite; }`

## 12. 금지 사항

1. ❌ 밝은 배경(흰색/회색) 위에 브랜드 색상 사용
2. ❌ 로고 색상 임의 변경
3. ❌ Purple Deep를 넓은 면적의 배경색으로 사용
4. ❌ 본문에 Gowun Dodum 서체 사용 (제목 전용)
5. ❌ 이모지를 3개 이상 연속 사용
6. ❌ "AI"라는 단어를 사용자 대면 텍스트에 사용
7. ❌ 불릿포인트(■●✦) 스타일의 기계적 텍스트
